数论分块的模板，附带一些小证明

 

数论分块，一般是用来求类似 \(\displaystyle\sum_{i=1}^{n} (f(i) \times \lfloor \frac{n}{i} \rfloor)\) 的和式。

他为什么可以用分块做？因为 \(\lfloor \frac{n}{i}\rfloor\) 一般只有最多 \(2\lfloor\sqrt{n}\rfloor\) 个值。如何证明？

  • 当 \(1 \le i \le \lfloor\sqrt n\rfloor\) 时，因为 \(i\) 只有 \(\lfloor\sqrt n\rfloor\) 种情况，所以值最多就只有 \(\lfloor\sqrt n\rfloor\) 种。
  • 当 \(\lfloor \sqrt n\rfloor < i \le n\) 时，因为值域范围就只能为 \([1, \lfloor \sqrt n \rfloor)\)（因为上一种情况已经把值 \(\ge \lfloor\sqrt n\rfloor\) 的全取完了）。所以值也最多只有 \(\lfloor \sqrt n \rfloor\) 种。

因为值域最多就那么多种，所以就算是 \(n = 2 \times 10^{15}\) 也轻轻松松。

那么怎么确定当值为某个数时，他的 \(i\) 取的边界为多少？

也就是设 \(\lfloor \frac{n}{i}\rfloor = k\)，使 \(\lfloor \frac{n}{j}\rfloor = k\) 这个 \(\max_j\) 分别是多少？

假设 \(\lfloor \frac{n}{i}\rfloor = k\)，则 \(k \le \frac{n}{i}\)，所以 \(\frac{n}{k} \ge \frac{n}{\frac{n}{i}}\)。

则 \(\lfloor \frac{n}{k}\rfloor \ge \lfloor \frac{n}{\frac{n}{i}}\rfloor = \lfloor i \rfloor = i\)。

所以 \(i \le \lfloor \frac{n}{k}\rfloor\)。

所以 \(i_{\max} = \lfloor \frac{n}{k}\rfloor = \lfloor\frac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor\)。（其实就是 \(n / (n / i)\)）

于是只要确定了这个值域区间的左端点 \(l\)（就是上个区间的右端点的 \(r + 1\)），就能通过 n/(n/l) 来求得这个至于区间的 \(r\)。

然后这个区间的答案就是给 \(f(i)\) 做个前缀和，求个 \((sum_r - sum_{l-1}) \times (\lfloor \frac{n}{l}\rfloor)\) 就可以了。

于是，板子题就来了。

因为题目要求算 \(\sum_{i=X}^{Y} f(i)\)，所以可以把它拆为 \(\sum_{i=1}^{Y} f(i)- \sum_{i=1}^{X} f(i)\)。

现在考虑 \(\sum_{i=1}^{n} f(i)\) 怎么求。

因为 \(f(i)\) 表示 \(i\) 所有约数的和，所以一个数 \(x\) 就可以对 \(f(x), f(2x), f(3x)\cdots\) 作出 \(x\) 的贡献。

而因为那些可以接收 \(x\) 产生的贡献的数一共有 \(\lfloor \frac{n}{x}\rfloor\) 个。所以做出的贡献一共就有 \(x\times \lfloor \frac{n}{x} \rfloor\)。

因为是分块整的，所以一次答案加上的就会有 \((l+(l+1)+(l+2)+\cdots+r) \times \lfloor \frac{n}{l}\rfloor\)。这个等差数列只需要用求和公式处理一下就行了。

学过小学数学的都知道，一个数 \(x\) 如果可以被质因数分解为 \(p_1^{k_1}p_2^{k_2}p_3^{k_3}\cdots p_n^{k_n}\)，那么他的因数个数就是 \((k_1+1)(k_2+1)\cdots(k_n+1)\)。如何证？

对于一个质因子 \(p_i\)，他可以选择把 \(m(0\le m \le k_i)\) 个 \(p_i\) 拿出来贡献给因数。也可以选择不贡献。如果所有质因子都贡献出全部了那最后的因数就是这个数本身，如果所有质因子都不贡献那么最后的因数就是 \(1\)。所以一个质因子可以有 \(k_i + 1\) 种情况。一共 \(n\) 个质因子，所以他的因数个数就是 \((k_1+1)(k_2+1)\cdots(k_n+1)\)。

就转化成和上一道题很相似的形式了。只不过贡献不是因数本身，是 \(1\)。所以更好做了。
