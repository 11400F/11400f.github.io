决策单调性分治

 

决策单调性分治，就是根据一个序列的决策单调性来分治找出序列中每个位置的答案。

什么是决策单调性？就是点 \(i\) 的决策点随着 \(i\) 递增也单调不降。根据这个单调性我们就可以先找到序列中间点的决策点，再左右分治。

用一道例题来说明什么是决策单调性。

P3515 Lightning Conductor

给定一个长度为 \(n\) 的序列 \(\{a_n\}\)，对于每个 \(i\in [1,n]\) ，求出一个最小的非负整数 \(p\) ，使得 \(\forall j\in[1,n]\)，都有 \(a_j\le a_i+p-\sqrt{|i-j|}\)。

\(1 \le n \le 5\times 10^{5}\)，\(0 \le a_i \le 10^{9}\) 。

思路：

要找到最小的非负整数 \(p\)，我们可以首先把 \(p\) 的表达式表示出来。

因为 \(a_j\le a_i+p-\sqrt{|i-j|}\)，移项，可得 \(p \ge a_j - a_i + \sqrt{|i-j|}\)。我们需要找对于每一个属于区间 \(1 \sim n\) 的 \(j\) 对应的这个 \(p\)，并且还要找 \(p\) 的最小值。那么就可以推出： \[ p_i \ge \displaystyle\max_{1\le j \le n}(a_j + \sqrt{|i-j|}) - a_i \] （因为 \(a_i\) 固定不变）。

但是这个绝对值看着很烦，于是我们把绝对值去掉，再先让 \(p\) 不保留整数（可以最后再取 ceil）： \[ p_i = \max \{\displaystyle\max_{1\le j \le i}(a_j + \sqrt{i-j}), \displaystyle\max_{i\le j \le n}(a_j + \sqrt{j-i})\} - a_i \] 总之，就是把他拆成两段分开算 max。这个只需要把区间 \(1\sim n\) 反转一遍再重新算然后取 max 就行了。这样就可以只考虑前半段： \[ p_i = \displaystyle\max_{1\le j \le i}(a_j - a_i + \sqrt{i-j}) \] OK。现在我们称每一个 \(j\) 为 \(i\) 的决策点，而使 \(p_i\) 能够到达最大值的决策点为 \(i\) 的最优决策点。我们设它为 \(pos_i\)。我们现在来求一下这个最优决策点的性质。

要从 \(i\) 转移到 \(i+1\)，先找两个点举例子。我们设有两个点 \(j\) 和 \(k\)，\(val_{j, i} = a_j - a_i + \sqrt{i-j}\)，现在我们要把这个东西转移到 \(val_{j, i+1} = a_j - a_{i+1} + \sqrt{i+1-j}\)。则他的转移量 \(\Delta val_j = a_j - a_{i+1} + \sqrt{i+1-j} - \sqrt{i-j}\) 。前面两项好比大小，但是最后的这个根号又该怎么算值呢？

可以数形结合：\(y = \sqrt{x}\) 的图像长什么样？

可以发现，当 \(x\) 越大，曲线就越平缓。可以推出来当 \(a < b\) 时， \(\sqrt{a} - \sqrt{a-1} > \sqrt{b} - \sqrt{b-1}\)。于是对比 \(\Delta val_j\) 和 \(\Delta val_k\)，发现前面的两项是可以消去的，若 \(j<k\)，则 \(i+1-j > i+1-k\)，则 \(\sqrt{i+1-j} - \sqrt{i-j} > \sqrt{i+1-k} - \sqrt{i-k}\)。

这时我们把 \(p_i\) 代入 \(k\) 点，我们可以发现，若 \(j < p_i\)，就算 \(p_{i+1} = p_i\)，最优决策点不动，\(j\) 也不如 \(p_i\) 优（因为 \(val_j < val_{p_i}\)）。所以第 \(i+1\) 个点对应的最优决策点 \(p_{i+1}\) 一定大于等于 \(p_i\)。然后 \(p_{i+2} \geq p_{i+1}\)，\(p_{i+3} \geq p_{i+2}\)……

发现了什么？\(p\) 序列有单调性！这就是前面所说的决策单调性。知道了 \(p\) 序列的决策单调性，就可以用前面的决策单调性分治：

  1. 首先设区间\([l, r]\) 的最佳决策点范围是 \([x, y]\)。
  2. 设一个变量 mid = l+r>>1，暴力扫区间 \(x \sim \min(mid, y)\)（取 min 是为了保证不会根号下负数） 找到它的最佳决策点 \(pos\)。
  3. 给 \(p_{pos}\) 赋值（取 max）。因为已经确定了 \(mid\) 的 \(p\) 值，所以将区间分为 \([l, mid-1]\) 和 \([mid+1, r]\) 继续分治。它们的最佳决策点范围分别是 \([x, pos]\) 和 \([pos, y]\)。
  4. 分治完了之后将区间反转，再分治一遍，再反转回来，即可得到答案。
