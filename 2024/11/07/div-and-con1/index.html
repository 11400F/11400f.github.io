决策单调性分治

11490DX Lv4
  2024-11-07 19:52:03 2024-11-07 19:52:03 Created   2024-11-09 09:16:43 2024-11-09 09:16:43 Updated  
  • C++ 
  • | 分治 
 

 

决策单调性分治，就是根据一个序列的决策单调性来分治找出序列中每个位置的答案。

什么是决策单调性？就是点 \(i\) 的决策点随着 \(i\) 递增也单调不降。根据这个单调性我们就可以先找到序列中间点的决策点，再左右分治。

用一道例题来说明什么是决策单调性。

P3515 Lightning Conductor

给定一个长度为 \(n\) 的序列 \(\{a_n\}\)，对于每个 \(i\in [1,n]\) ，求出一个最小的非负整数 \(p\) ，使得 \(\forall j\in[1,n]\)，都有 \(a_j\le a_i+p-\sqrt{|i-j|}\)。

\(1 \le n \le 5\times 10^{5}\)，\(0 \le a_i \le 10^{9}\) 。

思路：

要找到最小的非负整数 \(p\)，我们可以首先把 \(p\) 的表达式表示出来。

因为 \(a_j\le a_i+p-\sqrt{|i-j|}\)，移项，可得 \(p \ge a_j - a_i + \sqrt{|i-j|}\)。我们需要找对于每一个属于区间 \(1 \sim n\) 的 \(j\) 对应的这个 \(p\)，并且还要找 \(p\) 的最小值。那么就可以推出： \[ p_i \ge \displaystyle\max_{1\le j \le n}(a_j + \sqrt{|i-j|}) - a_i \] （因为 \(a_i\) 固定不变）。

但是这个绝对值看着很烦，于是我们把绝对值去掉，再先让 \(p\) 不保留整数（可以最后再取 ceil）： \[ p_i = \max \{\displaystyle\max_{1\le j \le i}(a_j + \sqrt{i-j}), \displaystyle\max_{i\le j \le n}(a_j + \sqrt{j-i})\} - a_i \] 总之，就是把他拆成两段分开算 max。这个只需要把区间 \(1\sim n\) 反转一遍再重新算然后取 max 就行了。这样就可以只考虑前半段： \[ p_i = \displaystyle\max_{1\le j \le i}(a_j - a_i + \sqrt{i-j}) \] OK。现在我们称每一个 \(j\) 为 \(i\) 的决策点，而使 \(p_i\) 能够到达最大值的决策点为 \(i\) 的最优决策点。我们设它为 \(pos_i\)。我们现在来求一下这个最优决策点的性质。

要从 \(i\) 转移到 \(i+1\)，先找两个点举例子。我们设有两个点 \(j\) 和 \(k\)，\(val_{j, i} = a_j - a_i + \sqrt{i-j}\)，现在我们要把这个东西转移到 \(val_{j, i+1} = a_j - a_{i+1} + \sqrt{i+1-j}\)。则他的转移量 \(\Delta val_j = a_j - a_{i+1} + \sqrt{i+1-j} - \sqrt{i-j}\) 。前面两项好比大小，但是最后的这个根号又该怎么算值呢？

可以数形结合：\(y = \sqrt{x}\) 的图像长什么样？

可以发现，当 \(x\) 越大，曲线就越平缓。可以推出来当 \(a < b\) 时， \(\sqrt{a} - \sqrt{a-1} > \sqrt{b} - \sqrt{b-1}\)。于是对比 \(\Delta val_j\) 和 \(\Delta val_k\)，发现前面的两项是可以消去的，若 \(j<k\)，则 \(i+1-j > i+1-k\)，则 \(\sqrt{i+1-j} - \sqrt{i-j} > \sqrt{i+1-k} - \sqrt{i-k}\)。

这时我们把 \(p_i\) 代入 \(k\) 点，我们可以发现，若 \(j < p_i\)，就算 \(p_{i+1} = p_i\)，最优决策点不动，\(j\) 也不如 \(p_i\) 优（因为 \(val_j < val_{p_i}\)）。所以第 \(i+1\) 个点对应的最优决策点 \(p_{i+1}\) 一定大于等于 \(p_i\)。然后 \(p_{i+2} \geq p_{i+1}\)，\(p_{i+3} \geq p_{i+2}\)……

发现了什么？\(p\) 序列有单调性！这就是前面所说的决策单调性。知道了 \(p\) 序列的决策单调性，就可以用前面的决策单调性分治：

  1. 首先设区间\([l, r]\) 的最佳决策点范围是 \([x, y]\)。
  2. 设一个变量 mid = l+r>>1，暴力扫区间 \(x \sim \min(mid, y)\)（取 min 是为了保证不会根号下负数） 找到它的最佳决策点 \(pos\)。
  3. 给 \(p_{pos}\) 赋值（取 max）。因为已经确定了 \(mid\) 的 \(p\) 值，所以将区间分为 \([l, mid-1]\) 和 \([mid+1, r]\) 继续分治。它们的最佳决策点范围分别是 \([x, pos]\) 和 \([pos, y]\)。
  4. 分治完了之后将区间反转，再分治一遍，再反转回来，即可得到答案。

P3140 Circular Barn Revisited G

虽然这道题不是一道决策单调性分治，但是它会为下一道题提供思路：

因为 \(n \le 100\)，所以可以想到可以 DP，先不考虑复杂度。

我们可以想新增一个入口会对后面的造成什么影响：

在新增了一个入口之后，在这个入口顺时针方向的这些牛就会缩短他和外界的距离，所以我们可以预处理出从一个打开入口 \(i\)，让 $i j $ 所有牛群都能进来的最小代价的数组 \(f\)：

于是我们可以根据这个来设状态：

\(dp_{i, k}\) 代表：现在从 \(1\) 扫到了 \(i\)，一共开了 \(k\) 扇门时使 \(1 \sim i\) 的牛全部回到原位所需的代价是多少。

转移方程： \[ dp_{i, k} = \displaystyle\min_{1\le j \le i}(dp_{j-1, k-1} + f_{j, i}) \] 也就是，我们枚举第 \(k\) 道门是在 \(j\) 位置开的，我们转移就把 \(1 \sim i\) 分为两段：

第一段 \(1\sim j-1\) 就可以直接从 \(dp_{j-1,k-1}\) 转移，就是前面一段，开了 \(k-1\) 道门，并且只扫到 \(j-1\) 的答案。

第二段 \(j \sim i\) 就是从 \(j\) 位置的入口进入把 \(j \sim i\) 的位置填满，让 \(j \sim i\) 的所有牛群都进来，他的代价就是 \(f_{j ,i}\)。

把他们两个相加就是这个状态的代价了。

但是由于原题是一个环，我们只能破环为链，枚举每一个起点，取所有起点的 \(dp_{终点,k}\) 的最小值。

时间复杂度 \(O(n^3k)\)，可以通过本题。

P6173 Circular Barn P

这道题与上一道题的不同之处就是：\(n\) 的范围变了。\(n \le 1000\)。上一道题的 \(O(n^3k)\) 无法通过。我们看怎么优化这个算法。

我们想一想，当 \(dp_{i, k}\) 的 \(i\) 变大时，使 \(dp_{i,k}\) 最小的那个决策点 \(pos_i\) 又是如何变化的？

感性理解一下，\(i\) 一直在变大，区间一直在变大。如果 \(pos_i\) 不变的话，那么右边的牛群就越来越多。这个时候只能将 \(pos\) 向右边移动来减小右边的这个变化量。所以当 \(i\) 递增，\(pos_i\) 也会单调不降。又是决策单调性优化！

然后还是枚举每一个起点，是 \(n\)，枚举每一个 \(k\)，就是 \(k\)，决策单调性分治，就是 \(n\log n\)。所以总复杂度就是 \(O(n^2k\log n)\)，可以通过本题。

P10967/P4767 邮局

前面是简单版，后面是加强版。

首先要求最简单的 DP 该怎么做。

如果你对某一个区间加了一个邮局，那么这个邮局该怎么放才能使区间到这个邮局的距离最小呢？我们发现这个就是一个绝对值方程： \[ \min(|X_l - pos| + |X_{l+1} - pos| + |X_{l+2} - pos| + \cdots + |X_{r} - pos|) \] 要求这个玩意的最小值，回顾初中数学，发现这个邮局的位置设在中位数就可以得到答案。所以我们可以处理这个中位数的 \(pre\) 和 \(suf\)（\(pre_{i, j}\) 就是 \(\sum_{k=i+1}^{j} X_{k} - X_i\)，\(suf_{i, j}\) 就是 \(\sum_{k=i-1}^{j} X_i - X_k\) ），然后把 \(pre_{mid, i}\) 和 \(suf_{mid, j}\) 加起来就可以得到在一个区间放一个邮局所需的最小距离了（可以自己手推式子）。

我们设刚刚说的最小距离为 \(dist_{i, j}\)，知道了这个玩意之后，就可以很好的推出 DP 式子了。

我们设 \(dp_{i, k}\) 为「考虑范围」为 \(1 \sim i\) 的所有村庄，设点 \(k\) 个邮局的最小距离总和（就是题目里让我们求的那个）。然后加上前面的 \(dist\)，我们可以得到： \[ dp_{i, k} = \min_{1\le j \le i}(dp_{j-1, k-1} + dist_{j, i}) \] 就是把这个 \(1\sim i\) 分成两部分：

第一部分是前面的 $1 j-1 $，是已知的 \(dp_{j-1, k-1}\)。

第二部分就是后面的 \(j \sim i\)，这一部分我们给他设一个邮局，然后所需的最小代价就是 \(dist_{j, i}\)。

边界就设 \(dp_{0, i} = 0\)，因为就算再多邮局来了，你也一个都管不到。

于是就可以用这种方法 DP 过简单版。时间复杂度为 \(O(V^2P)\)。

但是复杂版的 \(V = 3000, P = 300\)，不能过，考虑怎么优化。

假设我们之前在求 \(dp_{i, k}\) 的时候已经设好了最优的邮局选址在下标为 \(pos\) 的村庄，那么我们现在求 \(dp_{j, k}(j > i)\) 的时候，如果你还是选下标为 \(pos\) 的村庄不动，那么你右边的村庄就会越来越多，这个时候你只能把 \(pos\) 往右移动以达到距离最小。所以发现，当 \(i\) 增加的时候，\(pos\) 也在单调不降。说明可以用决策单调性分治来做。

时间复杂度 \(O(PV\log V)\)。
